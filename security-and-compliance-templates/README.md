# Security and Compliance Templates

## Overview

This repository contains practical templates and structured documentation examples related to **cybersecurity governance, risk management, and compliance**.

The purpose is to provide reusable frameworks that can help organizations structure their security and compliance processes.

The templates reflect an interest in the intersection between:

- cybersecurity
- governance
- compliance
- risk management
- regulatory requirements

---

## Contents

The repository includes templates for:

- cyber risk assessments
- incident response documentation
- security policies
- vendor and outsourcing security reviews
- regulatory compliance checklists
- security governance documentation

---

## Objectives

The goal is to:

- translate regulatory requirements into practical documentation
- demonstrate how organizations can structure security governance
- provide reusable frameworks for security and compliance work
- support structured and transparent security processes

---

## Regulatory Context

Many of the templates are inspired by requirements found in:

- NIS2 Directive
- DORA
- GDPR security obligations
- Cybersecurity Act
- Cyber Resilience Act
